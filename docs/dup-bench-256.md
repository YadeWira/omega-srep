# Omega SREP `-dup` benchmark (F5.4)

Generated by `tests/dup_bench.sh` on 2026-04-27T06:03:38Z.

## Setup

- Host: `Linux 6.12.74+deb13+1-amd64 x86_64`, 4 cores, 7.0 GiB RAM
- Method: `-m4`
- CDC buffer size: 8 MiB (`--buf=8388608`)
- Input: 32 MiB block × 8 = 256 MiB total
- Block content: 1 MiB text + 31 MiB xorshift64 binary, repeated identically

RSS measurements use `tests/measure.py` (a Python wrapper
around `getrusage(RUSAGE_CHILDREN)`, since GNU `time` isn't
installed on this host) and cover only the `osrep` invocation.
The F5.3a wrapper's split-encode and split-decode steps are
excluded — they read the full input into a contiguous buffer
and would dominate the comparison; F5.3b's native integration
will stream and remove that overhead.
